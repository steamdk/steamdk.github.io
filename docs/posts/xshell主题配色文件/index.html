Xshell突出显示配置#

  1. 将命名为 XshellColor.hls 的文件准备好。

  2. 将该文件放置到以下目录： C:\Users\Your_name\Documents\NetSarang Computer\7\Xshell\HighlightSet Files

    （请将 Your_name 替换为您的实际用户名）

  3. 打开 Xshell 后，即可在颜色设置中选择该高亮配置文件。

  4. 我这是黑暗版风格, 偏护眼
