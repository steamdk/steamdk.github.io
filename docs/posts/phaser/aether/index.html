项目介绍#

  • 一个 Phaser 3 开发的 RPG闯关游戏，通用化场景地图，怪物坐标 掉落回血物品及坐标等，支持小白用户使用 Tiled 自定义 各个关卡。
  • 目前是 aether 项目的改版，还处于开发阶段

实现方案#

  • 采用 phaser3 + ts 混合 js 语言编写
  • 采用 Tiled 编辑地图，以及行走区域
  • 采用 rollup 打包
