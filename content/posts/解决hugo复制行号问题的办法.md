# 问题

我的网站在复制代码时，会把行号也一起复制下来。我尝试了很多办法，但最终决定：

> 既然解决不了问题，那就干脆消灭问题吧！😆



# 解决办法

 在`hugo.yaml`文件直接增加以下代码, 这段代码的作用是禁用代码块中的行号显示。

```yaml
markup:
  highlight:
    lineNumbers: false
```
